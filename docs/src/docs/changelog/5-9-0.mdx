## Input.Placeholder component

`Input.Placeholder` component can be used to add placeholder to [Input and InputBase](https://mantine.dev/core/input/) components that are based on `button` element
or do not support placeholder property natively:

<Demo data={InputDemos.placeholder} />

## Other changes

- [RangeSlider](https://mantine.dev/core/slider/) component now supports `maxRange` prop
- [Stepper](https://mantine.dev/core/stepper/) component now supports any CSS color value in `color` prop
